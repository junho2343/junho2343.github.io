## 직종 : 웹디자인 및 개발

지방 기능경기 대회 결과까지 써버렸지만, 간단하게 저희 직종에 대해 설명을 하려 합니다.
(제가 대회를 치룬 2018년도 기준입니다.)

먼저 4가지의 과제가 존재하며, 각각 25점씩 100점 만점으로 상대평가를 진행하게 됩니다.
진행시간은 각 과제별 3시간씩 총 12시간이 주어집니다.

### 1과제 - Design

> 주어진 이미지 파일, 텍스트 등을 활용하여 각자 포토샵을 통해 웹 페이지를 제작하는게 1과제 입니다.
>
> 대회 전 보통 자신만의 웹 페이지 구성을 미리 생각 해놓은 뒤 대회 당일 제공받은 파일, 텍스트를 확인하여 주어진 기술서를 바탕으로 변형을 합니다.
>
> 주어진 기술서가 곧 채점항목 이기에 정말 꼼꼼히 보면서 진행을 해야합니다.
>
> 전국대회에서는 1점 차이가 엄청난 차이며, 소수점 단위의 점수를 조금이라도 더 확보하기 위해 디자인에서 보통 판가름이 많이 나는 편입니다.
> (2,3,4 과제와 달리 주관 점수가 존재하기 때문)

### 2과제 - Publishing

> 2과제는 완성된 포토샵 디자인 파일, 애니메이션 파일 등을 제공받은 뒤 기술서를 바탕으로 오직 html, css 를 활용하여 제작을 하게 됩니다.
>
> 일반적으로는 js로 동적인 애니메이션을 구현하지만, 2과제에서는 html, css 를 제외하고는 사용이 불가 하기에 css 만으로도 이만큼 구현이 가능하다는 걸 준비하면서 깨달았습니다.. (효율적 이지는 않..)
>
> 그래도 2과제를 준비하면서 퍼블리싱 속도, 반응형 디자인, 웹표준 등 거의 퍼블리싱에 필요한 모든 것들을 터득하여 간단히 퍼블리싱 외주도 받아서 용돈을 챙기기도 했습니다.

### 3과제 Client-side Development

> 3과제 부터는 기술서가 더욱 자세해 지기 시작합니다.
> 문제에 따라서 영역이 많이 달라지며 javascript 만으로 주어진 기술서에 맞게 개발을 해야합니다.
>
> 대표적으로 기억나는 문제는 영수증 관리, 이미지 포토샵, 게시물 관리, 애니메이션 떡칠 등이 있으며, 파일 업로드/다운로드, 캔버스, IndexedDB, 애니메이션 떡칠 등의 기능을 구현했던 것 같습니다.
>
> 보면은 일반적으로는 server 영역에서 구현해야 할 것들을 client 영역 문제로 내기도 하는데 실무에서는 쓰지도 않을 대회를 위한 기능 같았습니다.
>
> 그래도 덕분에 별에별 기능들을 구현하며 js 숙련도 또한 엄청나게 늘은 것을 나중에 체감할 수 있었습니다.

### 4과제 Server-side Development

> 마지막으로 server 영역은 php 를 기본 베이스로 하며 다른 과제와 똑같이 주어진 기술서에 맞게 기능을 구현하게 됩니다.
>
> 2, 3과제도 그렇고 4과제 또한 프레임워크나 라이브러리 등을 사용하지 않고 오로지 빈 소스에서 깡으로 개발을 합니다.
>
> 그렇다 보니 2, 3과제는 어찌 저찌 하더라도 4과제는 라우터 설정부터 Database 셋팅, 권한별 페이지 접근, 갖가지 기능 및 화면 렌더링 등등을 3시간 안에 개발을 해내야 합니다.
>
> 그래서 보통 시간이 남는 1,2 과제를 빨리 끝낸 뒤 4과제에서 사용 할 공통적인 라우터, 공통 함수 등을 미리 개발을 하여 시간을 조금 분배하는 방식으로 진행을 했습니다.
>
> 문제는 가장 대표적으로 블로그, 웹 드라이브, 배달 주문 웹 등등이 있고, 전국대회에서는 이런 문제에 알고리즘이 추가되어 제출 되기도 했었습니다.

확실히 프레임워크나 라이브러리 등을 사용하지 않고 직접 기능등을 구현하다 보니 훗날 회사에서 여러 프레임워크, 라이브러리를 사용할 때 더 도움이 되긴 했습니다.

## 전국 기능경기대회 준비

지방대회를 만족스럽게 마무리 하지 못한 아쉬움 때문에 전국대회는 약 5개월 전부터 정말 빡세게 준비를 했었던 것 같습니다.

문제를 정말 쉼없이 풀이 했고, 주말에는 양양 디지털 고등학교로 가서 아침부터 저녁까지 계속 서로 문제풀이를 하는 생활을 보냈었습니다.

이 과정에서 황준일쌤과 양양고의 최선한쌤 한테 정말 많이 배웠고 은혜를 입었습니다. 이 두분 덕분에 전국대회 메달을 땄다고 하더라도 과언이 아닐정도 입니다. 물론 이 두분을 저희에게 붙여주신 담당 영국쌤한테도 정말 감사드립니다.

준일쌤한테는 특히 단축키를 전수 받았는데 정말 개발 속도가 차원이 다르게 빨라졌고 지금도 애용하며 회사에서 단축키 마니아로 불리고 있습니다.

최선한쌤은 문제에 대한 알고리즘, js promise 등을 처음 배웠었고 그 이외에도 대회에 필요한 갖가지 노하우를 전수해 주셔서 정말 도움이 많이 되었습니다.

양양 디지털 고등학교는 경기지역에 속했는데, 경기는 서울과 달리 굉장히 빡센 지역이라 지방대회에서 부터 1점 차이로 메달이 바뀌는 지역 이였습니다.

그렇다보니 제가 느끼기 에는 동학년, 후배들 수준이 조금 남다르게 느껴졌습니다. (찐 개발자들 ㄷㄷ..)
